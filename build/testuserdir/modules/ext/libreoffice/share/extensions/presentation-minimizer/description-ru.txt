Presentation Minimizer используется для уменьшения размера файла презентации. Изображения будут сжаты, а невостребованные данные удалены.
Presentation Minimizer может оптимизировать размер изображения за счёт качества. Презентации, предназначенные для демонстрации на экранах и проекторах, не требуют такого же высокого качества изображений, как презентации, предназначенные для печати.
